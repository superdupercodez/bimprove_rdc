GET!?!

You're not supposed to be here doing this in here!

Instead, you should do POST to this same URL ({{full_url}}) with your image attached. If things workout, for example following kind of JSON will be returned for the image attached below:

[{'class': 0,
  'confidence': 0.3176967502,
  'name': 'barrier',
  'xmax': 874.7608032227,
  'xmin': 0.0,
  'ymax': 521.9098510742,
  'ymin': 77.8828277588},}

]

The returned JSON structure may contain an array of detected objects and their respective bounding box parameters. In here

class - Detected object class index 

confidence - 0.0 - 1.0, confidence points of object detection 

name - Detected object class name

xmax - Bounding box right corners' x coordinates on original image pixel space

xmin - Bounding box left corners' x coordinates on original image pixel space

ymax - Bounding box bottom corners' y coordinates on original image pixel space

ymin - Bounding box upper corners' y coordinates on original image pixel space
